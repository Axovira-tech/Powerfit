# POWERFIT Gym Website

A fully responsive, modern, and premium gym website built with HTML5, CSS3, and Vanilla JavaScript.

## Features

- **Modern UI/UX**: Dark mode aesthetic with energetic orange accents.
- **Responsive Design**: Mobile-first approach, fully optimized for tablets and desktops.
- **Animations**: Scroll reveal animations, hover effects, and smooth transitions.
- **Sections**:
  - Hero with animated background
  - Features grid
  - About Us with stats
  - Programs list
  - Trainer profiles
  - Membership pricing tables
  - Testimonial slider
  - Contact form with map placeholder

## How to Run

1. Open the `gym-website` folder.
2. Double-click `index.html` to open it in your web browser.
3. No server or build step required!

## Images

Note: Due to temporary service unavailability, placeholder images have been implemented using CSS gradients and icons. You can replace the `.placeholder-image` divs with actual `<img>` tags or background images in `style.css` when your assets are ready.

## Technologies Used

- HTML5 (Semantic)
- CSS3 (Variables, Flexbox, Grid, Animations)
- JavaScript (DOM Manipulation, Intersection Observer)
- FontAwesome (Icons)
- Google Fonts (Montserrat, Poppins, Inter)
